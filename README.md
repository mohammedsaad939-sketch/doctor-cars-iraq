## 🚀 الطريقة الأسهل: النشر المباشر (٥ دقائق، بدون أي برمجة)

### الخطوة ١: أنشئ حساب GitHub
اذهب إلى [github.com](https://github.com) وأنشئ حساباً مجانياً إذا لم يكن لديك واحد.

### الخطوة ٢: أنشئ مستودع (Repository) جديد
1. اضغط على **New repository**
2. اسمه: `doctor-cars-iraq`
3. اجعله **Public**
4. اضغط **Create repository**

### الخطوة ٣: رفع الملفات
في صفحة المستودع الجديد، اضغط **uploading an existing file** واسحب جميع ملفات هذا المشروع (بما فيها مجلد `src` كاملاً) ثم اضغط **Commit changes**.

### الخطوة ٤: النشر على Vercel
1. اذهب إلى [vercel.com](https://vercel.com) وسجّل دخول بحساب GitHub
2. اضغط **Add New Project**
3. اختر مستودع `doctor-cars-iraq`
4. Vercel سيكتشف أنه مشروع Vite تلقائياً — لا تغيّر أي إعداد
5. اضغط **Deploy**

بعد دقيقة واحدة تقريباً، ستحصل على رابط حقيقي مثل:
```
https://doctor-cars-iraq.vercel.app
```

شارك هذا الرابط مع أي شخص ليجرّب التطبيق من هاتفه أو حاسوبه مباشرة، بدون تثبيت أي شيء.

## 💻 التشغيل محلياً على جهازك (للتطوير)

يتطلب تثبيت [Node.js](https://nodejs.org) (النسخة 18 أو أحدث) أولاً.

```bash
# 1. افتح الطرفية (Terminal) داخل مجلد المشروع
cd doctor-cars-iraq

# 2. ثبّت المكتبات المطلوبة
npm install

# 3. شغّل المشروع محلياً
npm run dev
```

سيعطيك رابطاً محلياً مثل `http://localhost:5173` يمكنك فتحه في المتصفح لتجربة التطبيق فوراً.

## 📱 لتجربته كتطبيق هاتف (بدون متجر التطبيقات)

بعد نشره على Vercel، افتح الرابط من متصفح هاتفك (Safari على آيفون، أو Chrome على أندرويد):
- **آيفون:** اضغط زر المشاركة 📤 → "إضافة إلى الشاشة الرئيسية"
- **أندرويد:** اضغط القائمة ⋮ → "إضافة إلى الشاشة الرئيسية"

سيظهر التطبيق كأيقونة على هاتفك ويعمل بملء الشاشة مثل تطبيق حقيقي (هذا يسمى PWA).
